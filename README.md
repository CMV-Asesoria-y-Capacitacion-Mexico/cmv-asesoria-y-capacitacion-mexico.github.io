# CMV Asesoría y Capacitación México

Proyecto desarrollado como parte de servicio social.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Descripción
Desarrollo de sitio web institucional para fortalecer la presencia digital de la empresa y facilitar el contacto con clientes.

**Desarrollado por:** Gisela Cruz Anastacio

## Estructura del sitio
- Página única (one-page) con secciones: Hero, Servicios, Catálogo de cursos, ¿Quiénes somos?
- Footer informativo
- Botón flotante de contacto por Gmail

## Contacto y privacidad
Por instrucción expresa del cliente y por razones de **privacidad y seguridad**:
-  No se publica ningún número telefónico personal.
-  El único canal de contacto visible es el **botón flotante que abre Gmail** (correo electrónico).
-  Los mensajes enviados llegan directamente al correo oficial de la empresa.

Esta decisión garantiza:
- Trazabilidad de las comunicaciones.
- Evita llamadas no deseadas o fuera de horario.
- Protege los datos personales del cliente.

## Funcionalidades destacadas
- **Botón flotante**: Enlace `mailto:` que abre el cliente de correo predeterminado (Gmail u otro).
- **Footer**: Contiene información legal y derechos reservados (sin teléfonos).
- **Diseño responsive**: Adaptable a dispositivos móviles y tabletas.
